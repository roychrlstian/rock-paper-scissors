# Rock Paper Scissors

A simple browser-based Rock Paper Scissors game built with HTML, CSS, and JavaScript. Choose rock, paper, or scissors, then play against the computer and see the result instantly.

## Features

- Clean and lightweight interface
- Random computer choices each round
- Instant win/lose/draw feedback

## How to Play

1. Open `index.html` in your browser.
2. Select Rock, Paper, or Scissors.
3. Compare your choice with the computer's result.

## Tech Stack

- HTML
- CSS
- JavaScript
